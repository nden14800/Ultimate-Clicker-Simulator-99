# Ultimate Clicker Simulator 99

⛔ **開発終了 / Discontinued**

## 概要
クリッカー系シミュレーションゲームの実験プロジェクトです。
Cloudflare Pagesでの公開は終了しました。

## 終了の理由
ゲーム内の数値インフレが制御不能となり、進行不可能なバグ（単位表示の崩壊、数値のオーバーフロー）が頻発したためです。
特にゲーム後半において、数値の桁数がプログラムの処理限界を超え、正常なゲームプレイが不可能となりました。

これ以上のバランス調整は困難と判断し、開発を断念しました。
